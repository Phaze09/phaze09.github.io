CSS Color

CSS uses color to control the appearance of text, backgrounds, borders, and many other visual elements on a webpage. By using different color values, designers can improve readability, create contrast, establish branding, and guide users’ attention. CSS provides multiple properties and multiple formats for defining color, giving developers flexibility and precision.

CSS Color Properties

color

The color property changes the color of text inside of element. It affects the foreground color of the content, such as paragraphs, headings, and links.

background-color

The background-color property sets the background color of an element. It fills the space behind the content and padding.

border-color

The border-color property sets the color of an element's border. It only works if a border style and width have already been defined.

How CSS Accepts Color Values

CSS accepts several different formats for defining color values.

Named Colors

CSS includes predefined color names.

Examples: red, green, purple, black, white.

Hexadecimal Colors

Hex colors begin with # followed by six characters (numbers 0-9 and letters A-F). They represent red, green, and blue values.

Example: #FF5733 (FF represents red, 57 represents green, and 33 represents blue).

RGB Colors

RGB stands for Red, Green, and Blue. Each value ranges from 0 to 255.

Example: rgb(255, 87, 51) which is the same color as the one above.

HSL

HSL stands for Hue, Saturation, and Lightness. It uses degress for hue (0-360) and percentages for saturation and lightness.

Example: hsl(9, 100%, 60%) which is the same color as the two above.
