Code Log 2: CSS Measuring Units

Absolute vs Relative

Absolute Length Units are fixed and don't change based on the screen size, parent elements, or user settings. They represent a specific physical or fixed size.

Relative Length Unites change dependinng on something else like the parent element, the root element, or the viewport size.

Describing Units

px (Pixels)

  • Abosolute Unit
  • Represents one unit of screen display resolution
  • Not always a physical screen pixel
  • Easy to understand
  • Doesn't automatically scale with user font preferences
