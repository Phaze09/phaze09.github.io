Assignment Questions

What is the difference between non-semantic and semantic tags in HTML?

  • Non-semantic HTML tags don't describe or exaplin what the content means, they are there to apply styling and structure.
  • Semantic HTML tags are clear and describe to purpose of the content inside them.

What are two (or more) benefits of using semantic tags?

  • Semantic tags give clear information about the structure of the page. They can help users navigate a site so much easier.
  • Apparently, search engines understand semantic tags better because it helps them figure out what content is the most important.
